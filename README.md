# TPCRP Active Learning on CIFAR-10

This repository contains an implementation of the TPCRP active learning strategy applied to the CIFAR-10 dataset, based on the paper "Active Learning on a Budget: Opposite Strategies Suit High and Low Budgets".

## Prerequisites

Please download and use **Python 3.12.x** specifically for creating your virtual environment and running this project.

## Setup Instructions

It is highly recommended to run these notebooks inside an isolated Python virtual environment. 

1. Create a virtual environment named `.venv` in the root of your project directory using Python 3.12.x:
```bash
python3.12 -m venv .venv
```

2. Activate the virtual environment:
* On macOS and Linux:
  ```bash
  source .venv/bin/activate
  ```
* On Windows:
  ```bash
  .venv\Scripts\activate
  ```

3. Register the virtual environment as a Jupyter kernel and install the necessary dependencies:
```bash
pip install ipykernel torch torchvision scikit-learn matplotlib plotly pandas
pip install -r requirements.txt
python -m ipykernel install --user --name=.venv --display-name="Python (.venv)
```

 

## Execution Order

The notebooks are designed to be run sequentially. The second notebook analyzes and plots data generated by the first one.

1. **First, run tpcrp_cifar10.ipynb**
   Open this notebook, select the `.venv` kernel, and run all cells. This script runs the primary active learning loops, evaluates different query strategies (TPCRP, Uncertainty, Margin, Entropy), and saves the results.

2. **Next, run modified_tpcrp_cifar10.ipynb**
   You must run this notebook strictly after the first one has completely finished. This modified version processes the saved data to analyze class imbalance across different budgets and generates the final Total Variation (TV) distance plots.
